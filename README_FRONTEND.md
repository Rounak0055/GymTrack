## 🚀 3-Step Quick Start

### Step 1: Install Dependencies
```bash
cd gym-frontend/gym-frontend
npm install
```

### Step 2: Start Backend (separate terminal)
```bash
cd gym-backend
mvn spring-boot:run
```
Wait for: `Tomcat started on port(s): 8080`

### Step 3: Start Frontend
```bash
npm start
```
Opens: **http://localhost:3000**
